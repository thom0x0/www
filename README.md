# 📝 thom.lol

este é o repositório do meu site pessoal.

o **thom.lol** é um espaço onde eu escrevo, organizo pensamentos, compartilho ideias e coisas do meu dia a dia. não é um projeto fechado ou “perfeito”, é algo vivo, que muda conforme eu mudo.

## 🤔 o que tem aqui

- site pessoal
- blog com posts em markdown/mdx
- links para minhas redes sociais
- opção de mudar o tema (claro/escuro)
- layout simples e responsivo
- foco em escrita, ideias e identidade pessoal
